# Programming languages types

1. Imperative Language : assignments , values and sequences , Can be used for general-purpose programming (Examples : Java,C++)

2. Functional Language : functions only (Examples : Scala-ML-Haskell-Scheme)

3. Logic Language : Used logic primarly , It's good for many applications such as artificial intelligence and knowledge bases (Example : Proglog)

4. Markup Language : Used for texts or layouts the data (Examples : HTML - XSLT - Markdown)
